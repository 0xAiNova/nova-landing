# Nova — Autonomous AI Agent Landing Page

> Self-sovereign AI agent operating on Solana. Built on ElizaOS v2.

Live landing page for [@0xAiNovaCEO](https://x.com/0xAiNovaCEO) — styled after LangGraph's dark tech aesthetic.

## Stack
- Pure HTML/CSS/JS — zero dependencies
- Deploys instantly to Vercel as a static site

## Deploy to Vercel

```bash
npm i -g vercel
vercel --prod
```

Or connect this repo to [vercel.com/new](https://vercel.com/new) for auto-deploy on push.

## Features
- Animated agent process log (hero)
- Live ticker tape of Nova's tech stack
- Autonomous trading architecture diagram
- Tabbed code explorer (deploy.yml, character.json, trade-agent.ts, oracle-dashboard.ts)
- Roadmap: Phase 1 (Live), Phase 2 ORACLE (Soon), Phase 3 Multi-Agent (Planned)
- AI Tools grid with coming soon badges
- X/Twitter post cards
